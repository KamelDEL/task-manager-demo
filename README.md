# Kanban Task Manager

A powerful drag-and-drop task management tool for guiding users through organizing workflows naturally.

## Features

- **Drag & Drop Interface:** Full Kanban board functionality powered by @hello-pangea/dnd. Move tasks between columns effortlessly.
- **Task Management:**
  - **Create:** Add new tasks with details like content, priority, and due dates.
  - **Edit:** Click the edit icon to modify existing tasks in a modal dialog.
  - **Delete:** Intuitive "Drag to Trash" interaction - simply drag a task to the bottom-right bin to remove it.
- **Priority System:** Visual color-coding for High, Medium, and Low priority tasks.
- **Persistence:** Board state (columns and tasks) is automatically saved to LocalStorage.
- **Theming:** Full support for Dark and Light themes.

## Getting Started

1. Install dependencies:

   npm install


2. Run the development server:

   npm run dev


Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
